## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/nika-partsvania/Memory-Game
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start 
   ```sh
   npm start
   ```
